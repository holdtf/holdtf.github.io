Экономьте с GitHub Pages.

GitHub - это сервис для разработчиков программного обеспечения. Он позволяет сохранять файлы на сервере и обмениваться ими с другими людьми. Существуют скрытые (платные) репозитории, и открытые (бесплатные). Если в корневой папке репозитория разместить файл index.html, то получится главная страница сайта GitHub Pages.

Бесплатный хостинг мы уже получили. Также абсолютно даром приобретаем поддоменное имя (имя аккаунта на GitHub) и SSL сертификат. Цены на всё сэкономленное вы можете поискать в сети.
